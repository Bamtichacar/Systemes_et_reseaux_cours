# Installation Ubuntu Server dans VirtualBox

Installation d'un serveur Linux avec VirtualBox.

## Outils utilisés

- VirtualBox
- Ubuntu Server

## Configuration de la machine virtuelle

Nom : ubuntu-server  
RAM : 2048 MB  
Disque : 20 GB  

## Installation

1. Télécharger Ubuntu Server
2. Créer une machine virtuelle Linux
3. Monter le fichier ISO
4. Démarrer l'installation

## Utilisateur créé

Server name : ubuntu-server  
Username : user1  

## Objectif

Utiliser ce serveur pour apprendre :

- commandes Linux
- administration système
- gestion utilisateurs
